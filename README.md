# Soap Bubble Project

## Pre-requisites

### Hardware

- ESP32CAM
- USB-UART adapter
- USB cable
- Dupont cables

### Software

- VSCode
- PlatformIO
- Kubernetes Cluster (available with Orbstack or Docker Desktop)
- Golang
- kubectl
- kubebuilder
